# Basics - Fundamentals Review

Work through these in order. Each file has empty functions and test cases.
Run each file with `python3 <filename>.py` — tests will tell you if you got it right.

## Order

1. **01_array_traversal/** - Looping, indexing, basic iteration
2. **02_array_manipulation/** - Modifying arrays in place, slicing, building new arrays
3. **03_string_basics/** - Iteration, building strings, char operations
4. **04_dict_basics/** - Counting, grouping, lookups
5. **05_two_pointers/** - Left/right pointer patterns
6. **06_sorting_searching/** - Basic sorts, binary search
7. **07_stacks_queues/** - LIFO/FIFO patterns
8. **08_sets/** - Uniqueness, intersections, membership

Once you're comfortable with all of these, the problems in Arrays/, Strings/, and Hash Tables/ should feel natural.
